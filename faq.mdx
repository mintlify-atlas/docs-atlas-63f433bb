Find answers to common questions about using the ChatGPT Scraper API.

<Accordion title="What's the recommended timeout?">
  We don't recommend putting any timeout, given that our system retries automatically. We recommend setting up a retry mechanism in case of failure.
  
  However, if you must set a timeout, use 120-180 seconds as ChatGPT responses may take time to generate depending on complexity.
</Accordion>

<Accordion title="How does credit usage work?">
  The base ChatGPT scraper request uses **5 credits**. Additional features have extra costs:
  
  - `include.rawResponse`: +2 credits per request
  - `include.searchQueries`: +2 credits per request
  
  You can combine multiple features, and credits are added together. For example, enabling both `rawResponse` and `searchQueries` would cost 9 credits total (5 base + 2 + 2).
</Accordion>

<Accordion title="Do shopping cards require extra credits?">
  No, shopping cards are automatically extracted when available at no additional cost. They're included by default in the response structure.
  
  When ChatGPT returns product or commercial information, cloro automatically extracts comprehensive shopping data including product titles, prices, ratings, merchant information, and multiple offers - all at no extra charge.
</Accordion>

<Accordion title="What formats can I get the response in?">
  The ChatGPT Scraper API returns structured JSON by default, which includes:
  
  - Plain text response (`result.text`)
  - Optional Markdown format (`result.markdown` when `include.markdown: true`)
  - Shopping cards when available
  - Model information
  - Optional raw response and search queries
  
  The Markdown format is particularly useful for integration with AI tools and documentation workflows.
</Accordion>

<Accordion title="What are shopping cards?">
  Shopping cards are structured product data automatically extracted from ChatGPT responses when product or commercial information is included. Each shopping card contains:
  
  - Product titles, prices, and images
  - Ratings and review counts
  - Multiple merchant offers for price comparison
  - Stock and delivery information
  - Promotional tags (e.g., "Best price")
  - Rating citations and supporting websites
  
  This feature is unique to cloro's ChatGPT scraper and requires no additional parameters or credits.
</Accordion>
